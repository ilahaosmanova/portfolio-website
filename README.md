# Ilaha Osmanova – Personal Website

A personal portfolio website built with HTML and CSS, created for the SITE-1101 course at ADA University.

---

## Pages

### Home (`index.html`)
The landing page with a brief introduction, a profile photo, and links to my social/coding profiles:
- [GitHub](https://github.com/ilahaosmanova/personal-website)
- [Codecademy](https://www.codecademy.com/profiles/SITEstudent)
- [LinkedIn](https://www.linkedin.com/in/ilaha-osmanova-004886389/)

### About (`about.html`)
A detailed page covering three areas:
- **Academic Background** – my university studies at ADA University
- **Skills** – technical and soft skills I have developed
- **Activities & Hobbies** – things I enjoy outside of academics

### Projects (`projects.html`)
A showcase of three team projects completed during the SITE-1101 course:

| # | Project | Description |
|---|---------|-------------|
| 1 | Logic Gates | Built basic and complex logic gate circuits in a university hardware lab |
| 2 | Hour of Code | Taught basic coding to 7th-grade students using Microsoft MakeCode |
| 3 | Robotics | Programmed a LEGO Mindstorms robot to draw geometric figures |

Each project includes a title, description, photo, and a link to a YouTube video.

### Game (`game.html`)
A classic Snake game built with HTML5 Canvas and JavaScript. Use the arrow keys to control the snake and try to beat your high score!

---

## Built With

- CSS3 (custom properties, CSS Grid, responsive design)
